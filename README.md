# Instruções
Clone o Repositório:<br>
`git clone <link repositório>`<br>

Entre na pasta da aplicação:<br>
`$ cd todo-list`<br>

Inicie o Servidor de Desenvolvimento:<br>
`npm run dev`<br>
`yarn run dev`<br>
